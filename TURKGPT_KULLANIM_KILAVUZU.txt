❓ SORUN GİDERME
==============================================

HTML: "401 API hatası"
→ Düzeltildi! Yeni HTML dosyasını indirin
→ İnternet bağlantınızı kontrol edin

HTML: Çalışmıyor
→ Modern tarayıcı kullanın (Chrome, Firefox, Edge)
→ JavaScript'in etkin olduğundan emin olun
→ Konsolu açın (F12) ve hataları kontrol edin

Python Desktop: "Module not found"
→ pip install requests
→ Python 3.8+ yüklü olmalı

EXE: Çalışmıyor
→ Windows Defender'dan izin verin
→ Antivirüs ayarlarını kontrol edin

API Yavaş
→ İnternet bağlantınızı kontrol edin
→ Normal yanıt süresi: 2-5 saniye
